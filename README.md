# ReCAST – Research Center for Applied Sun Tzu Bin Fa and Taijiquan

ReCAST 致力於結合孫子兵法的戰略哲學、太極拳的具身實踐，以及當代管理與教育的應用，推動東方智慧的現代化詮釋與跨域落地。
